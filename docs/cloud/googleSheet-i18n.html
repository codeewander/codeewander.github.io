使用 GoogleSheet 產生 i18n 文件

需求情境：1. 多人共同編輯翻譯內容 2. 能夠快速新增語系和自動化翻譯 3. 有授權的人才可以編輯 作法：將翻譯內容放在 Google Sheet 文件中，並串接 Google Sheet API 將翻譯下載成JSON檔案

權限處理 - 使用 Google Service Account 設定 Google sheet API 權限#

步驟 1 : 到 Google Cloud 頁面新增專案 步驟 2: 到API和服務頁面，並啟用 Google Sheets API 步驟 3: 到 API和服務 下面的憑證點擊建立憑證，然後選擇服務帳戶，把欲授權的 email 帳號加上去 步驟 4: 點擊 金鑰，然後點擊建立新的金鑰，輸出 JSON 格式，放進專案資料夾

Google Sheet 設定#

第一欄設定為 Key ，而後面是欲翻譯的語系代碼
